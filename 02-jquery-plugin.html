Extending jQuery with a Plugin

Plugins are additional pre-written scripts written by other developers or yourself that extend the capabilities of jQuery.

jQuery plugins are usually attached to an HTML page after the jQuery library but before any custom jQuery scripts.

Plugins can do almost anything from creating slideshows to validating forms to extending jQuery's animation capabilities.

Using the Cycle2 jQuery Plugin

The Cycle2 jQuery plugin is a very simple plugin to use, with lots of configuration options. The plugin has excellent documentation. You can download the plugin and read about how to use the Cycle2 plugin at http://jquery.malsup.com/cycle2/.

This plugin can use classes and data attributes for its configuration, meaning you do not have to write any javaScript code to get it to work.
